## 오늘 한 일
📑 **API 설계**
처음으로 백엔드와 함께 API 설계를 했다. 근 2주동안 백엔드와 함께 협업 하면서 작업을 할 때 필요한 데이터들이 있었고 그걸 받지 못해서 작업을 하지 못한 적이 있었다. 그래서 이번에는 6주동안 실전 프로젝트를 진행 하면서 받을 건 꼭 받아서 작업을 하고 완벽하게 프로젝트를 만들고 싶어 백엔드 분들과 몇시간을 걸쳐서 API 설계를 함께 했다. 

[API 설계](https://www.notion.so/7d6d2ec09dd14b72aaa03212cbd8f2ca?v=7d823be4229e45ebb042d2182900c504)

그동안 로그인, 회원가입, 게시글 CRUD, 후기 CRUD, 개인 페이지등을 하면서 꼭 필요한 데이터들을 요구 했고 가능하다면 상태코드를 보내주면서 message도 같이 보내주면 좋을 것 같다고 말씀을 드렸는데 착하신 팀원들은 다 할 수 있다면서 보내주신다고 하셨다. 덕분에 완벽하게 체크할 수 있을 것 같다. 
오늘은 API 설계와 처음 하는 기능들(좋아요, 북마크, 지도, 이미지 보내기, 무한스크롤)을 살펴보았고 디자이너분에게 와이어프레임을 받으면 그때부터 작업을 시작하기로 했다
