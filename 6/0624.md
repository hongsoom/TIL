## 오늘 한 일
항해99 6조 실전프로젝트 - 여행 사이트(사이트 이름 미정)

6월 24일부터 시작한 실전 프로젝트! 25~26일 이틀 동안은 어떤 사이트를 만들지, 어떤 기술을 쓸 지, 어떤 식으로 보여줄 지에 대해 기획을 했다.
오랜 회의 끝에 여행 계획을 세우는데 불필요하게 많은 시간소모와 스트레스를 받거나 여행 계획을 세우는데 익숙하지 않거나 미리 경험해본 여행자들의 정보를 알고 싶은 사람들을 위해 여행 경험자들의 여행 코스와 정보를 공유하는 서비스를 만들어 보도록 결정했다.
후에, 타켓층을
- 여행 계획을 세우기에 시간적 여유가 없는 사람들
- 여행 계획을 세우기 귀찮은 사람들
- 다른 사람들의 여행 계획을 보고 싶은 사람들
으로 정했다.

마지막으로 주요 기능으로
- 통합 커뮤니티(지역별 태그/필터)
- 게시판 카테고리 → 지역별(서울, 광역시, 세종 , 경기도, 강원도, 충청도, 경상도, 전라도, 제주도) → 테마별(액티비티, 먹거리, 볼거리, 힐링)
- 소셜 로그인
- 메세지 기능
- 좋아요 기능
- 북마크 기능
- 카카오톡 공유 기능
- 카카오맵 동선 표시 기능
- 게시글 CRUD
- 후기 CRUD
- 베스트 게시글
- 검색 기능
으로 정했고 모두 성공적으로 기능을 구현을 한다면 추가 기능으로 
- USER 팔로우 기능
- 게시글 많이 작성한 user에게 등급 부여
를 하기로 했다.

아직 와이어프레임이 만들어지지 않아서 바로 코딩을 시작하기에 어려움이 있지만 어떤 기능을 할 것 인지, 어떤 사이트를 할지 정확히 결정을 하게 되서 찝찝하지만 만족한 하루였다.
