**(3) 아이콘 사용하기**

그리고 코드에 import하면은 아이콘 이름으로 사용할 수 있다. 사이즈 조절도 됨! ![](https://velog.velcdn.com/images/hongsoom/post/ad171227-e505-4ffc-a0d1-7ec4029bfd68/image.PNG)

**(3) 아이콘 고정시키기**
아이콘들을 오른쪽 정렬로 했는데 글자가 길어질 수록 아이콘들이 밀려나기 시작했다. 그래서 아이콘들의 위치를 고정시키려고 한다.
```javascript
position: absolute;
right: 12px;

position: absolute;
right: 45px;

position: absolute;
right: 80px;

```
**CSS Position** :  문서 상에 요소를 배치하는 방법
**top, right, bottom, left** 속성을 통해 요소의 최종 위치를 결정

**block** : 한 줄 모두 차지 (대표 element - < div >, < p > )
**inline** : 콘텐츠 크기 만큼만 차지 (대표 element - < span >)

|값 |	의미|
|---|---|
static	|기준 없음 (배치 불가능 / 기본값)
relative	|요소 자기 자신을 기준으로 배치
absolute	|부모(조상) 요소를 기준으로 배치
fixed	|뷰포트 기준으로 배치
stickey	|스크롤 영역 기준으로 배치

**Absolute**
요소를 일반적인 문서 흐름에서 제거
**가장 가까운 위치에 있는 조상 요소를 기준**으로 배치
조상 요소 위치를 기준으로 위쪽(top), 아래쪽(bottom), 왼쪽(left), 오른쪽(right)에서 **얼마만큼 떨어질 지 결정**
조상 중 Position을 가진 요소가 없다면 초기 컨테이닝 블록**(< bod>요소)를 기준**으로 (static을 제외한 값)
문서 상 원래 위치를 잃어버린다. (아래에 있는 div가 해당 자리를 차지한다)
