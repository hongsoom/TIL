## 오늘 한 일
### 📃 내 일주일 평점 남기기 
- **Main**  
  - 일주일 평점 보여주기 
  - 평점은 1~5까지 숫자 중 랜덤한 정수
  - 각 요일 옆 삼각형 버튼을 누르면 요일 평점 남기기 페이지로 이동하기
- **Scroe**  
  - 각 요일별 랜덤 정수로 구성된 점수의 평균을 구하기
  - Reset 버튼을 누르면 평점 평균을 0으로 상테변화 시키기
- **Detail** 
  - 선택한 요일 보여주기
  - 동그라미를 눌러서 평점 입력하기 (1번째 동그라미 누르면 1점, 3번째 동그라미 누르면 3점)
  - 남기기 버튼을 누르면 이전 페이지로 이동하기

![](https://velog.velcdn.com/images/hongsoom/post/bf28fc66-2b27-4d47-8dc6-9ad7734010bf/image.PNG)

1) 페이지 구성 : 컴포넌트를 3개로 나눔 (Main, Detail, Scroe)
2) 프로젝트 생성 : yarn create react-app my-week
3) App.js 내용 지우고 Main, Detail 경로 설정
```javascript
import {Route,Switch} from "react-router-dom";
import React from "react";
import Main from "./components/Main";
import Detail from './components/Detail';


const App = () => {

  return (
    <div className="App">
      <Switch>
        <Route path="/" exact>
          <Main  />
        </Route> 
        <Route path="/detail/:id" exact>
          <Detail  />
        </Route> 
      </Switch>
    </div>
  );
}

export default App;
```

- **useState**
함수 컴포넌트에서도 상태 관리를 할 수 있는 useState
```javascript
import React from "react";

const Nemo = (props) => {
  // count에는 state 값이, setCount는 count라는 state 값을 수정하는 함수가 될거예요.
  // useState(초기값): () 안에 초기값을 넣어줍니다.
  const [count, setCount] = React.useState(3);

  const addNemo = () => {
    // setCount를 통해 count에 저장된 값을 + 1 해줍니다.
    setCount(count + 1);
  };

  const removeNemo = () => {
    // setCount를 통해 count에 저장된 값을 - 1 해줍니다.
    // 이번엔 if문 대신 삼항 연산자로 해볼거예요!
    setCount(count > 0 ? count - 1 : 0);
  };

  const nemo_count = Array.from({ length: count }, (v, i) => i);
  // 반환할 리액트 요소가 없을 때는 null을 넘겨주세요!
  return (
    <div className="App">
      {nemo_count.map((num, idx) => {
        return (
          <div
            key={idx}
            style={{
              width: "150px",
              height: "150px",
              backgroundColor: "#ddd",
              margin: "10px",
            }}
          >
            nemo
          </div>
        );
      })}

      <div>
        {/* 함수를 호출합니다. */}
        <button onClick={addNemo}>하나 추가</button>
        <button onClick={removeNemo}>하나 빼기</button>
      </div>
    </div>
  );
};

export default Nemo;
```
- **useRef**
함수컴포넌트에서 ref를 쉽게 사용할 수 있도록 하는 함수
```javascript
import React from "react";
import styled from "styled-components";

const BucketList = ({ list }) => {
  const my_lists = list;
  const my_wrap = React.useRef(null);

  console.log(my_wrap); // 콘솔로 확인해봐요!

  window.setTimeout(() => { // 1초 뒤에는?!
    console.log(my_wrap);
  }, 1000);
  return (
    <div ref={my_wrap}>
      {my_lists.map((list, index) => {
        return <ItemStyle key={index}>{list}</ItemStyle>;
      })}
    </div>
  );
};

const ItemStyle = styled.div`
  padding: 16px;
  margin: 8px;
  background-color: aliceblue;
`;

export default BucketList;
```
- **Router**
브라우저 주소에 따라 다른 페이지를 보여주는 것 
```
Route 사용방법 1: 넘겨줄 props가 없을 때
<Route path="주소[/home 처럼 /와 주소를 적어요]" component={[보여줄 컴포넌트]}/>

Route 사용방법 2: 넘겨줄 props가 있을 때
<Route path="주소[/home 처럼 /와 주소를 적어요]" render={(props) => (<BucketList list={this.state.list} />)} />
```

## 오늘 느낀점
그냥 무작정 프로젝트를 만들어봤다. 마냥 앉아서 강의만 들을 수 없다고 생각한다. 강의 보는건 좋다. 하지만 이걸 보고 내가 응용할 수 있어야 하고 바로 사용할 줄 알아야 하는건데 그냥 보고만 있으면 생각 자체를 잘 안하게 되니까 그냥 시작해버렸다. 제일 먼저 했던건 어떻게 페이지를 보여줄까? App.js에선 경로 설정만 하고 다른 컴포넌트로 보여주기로 생각해서 첫 화면인 Main.js, 메인 화면 바로 아래에 평균 점수를 보여줄 수 있는 Scroe.js를 만들어 Main.js에 넣기로 하고 마지막으로 버튼을 누르면 해당 되는 요일 페이지로 넘어가는 Detail.js까지 컴포넌트를 세개를 만들어 봤다. 그리고 react-router-dom, styled-components 패키지를 깔고 App.js에 경로 설정까지 해주었다. 내일부터 해당 되는 페이지에 구현해야할 것들을 하나하나씩 해볼 것이다. 앞으로 막힐 순 있지만 열심히 구글링 하고 팀원들과 얘기하다보면은 답을 얻지 않을까?
