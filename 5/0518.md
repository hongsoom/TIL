- React 과제
 https://lucent-chimera-a69464.netlify.app
 
React를 본격적으로 공부하기전에 HTML 기본 태그를 쉽게 활용할 수 있도록 각 입력 양식 태그와 태그의 속성들을 공부했다.
```
<!doctype html>
<html>

<head>
    <title>React 과제</title>
</head>
<body>
    <form action="" onsubmit="alert('제출 완료!');">
        First name : <br />
        <input id="name" type="text" required> <br />
        Last name : <br />
        <input type="text" required> <br />
        Email : <br />
        <input type="text" required> <br />
        Gender :
        <input type="radio" name="male" value="Male" >
        Male
        <input type="radio" name="female" value="Female">
        Female
        <br />
        Favorite :
        <input type="checkbox" name="html" value="HTML" />HTML
        <input type="checkbox" name="java" value="JAVA" />JAVA
        <input type="checkbox" name="javascript" value="JavaScript" />JavaScript
        <br />
        Browser :
        <select id=browsers name="browsers">
            <option value="크롬" selected>크롬</option>
            <option value="사파리">사파리</option>
            <option value="엣지">엣지</option>
        </select>
        <br />
        Birthday :
        <input type="date" name="birthday" />
        <br />
        <input type="submit" value="제출"/> 
    </form>
</body>

</html>
```
사실 금방 코드를 짰다. 하지만 문제가 생겼다. 서버에 배포하려고 시도 했지만 자꾸 에러가 떴다.
