**4. 클릭 배열 만들고 상태변화함수 설정**

| click | index |
|------|---|
| 1 | 0 |
| 2 | 0,1|
| 3 | 0,1,2|
| 4 | 0,1,2,3|
| 5 | 0,1,2,3,4|

```javascript
const [click, setClick] = useState([0,1,2,3,4]);

setClick(click[i] + i) // 0만 나오고 나머지 NaN -> 빈배열일 때 undefined

setClick(click[i] + 1) // 1만 나오고 나머지 NaN -> 빈배열일 때 undefined

setClick(click + i) // 0,1,2,3,401234 -> 배열의 초깃값 없앰
```

그래서 배열의 초깃값을 없애고 클릭을 할때 상태변화함수에 index값을 넣어주기로 함

```javascript
setClick(i) // 0 ~ 4
```
